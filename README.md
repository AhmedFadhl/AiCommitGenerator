# AI Commit Generator (AJ)

This extension helps you generate commit messages using AI providers like Gemini and OpenAI.  
It also supports optional GitHub issue management.

## Features
- Generate commit messages from staged changes
- Select AI provider and model
- Optional GitHub issue creation
- Keyboard shortcut: `Ctrl + Alt + C`

## Configuration
See the extension settings under:  
**Settings → Extensions → AI Commit Generator**

## Usage
Open the Source Control panel and click **Generate AI Commit Message**.
